SCHOOLERS - IMAGE ASSETS DIRECTORY
====================================

Place the following images in this directory:

1. logo.png
   - Schoolers logo with trophy icon
   - Recommended size: 200x60px
   - Used in: Login page header, Sidebar, Navbar

2. kids-cartoon.png
   - Cartoon illustration of school kids/students
   - Recommended size: 400x400px
   - Used in: Left panel of all auth pages (Login, OTP, etc.)

3. student-avatar.png
   - Default student profile photo placeholder
   - Recommended size: 100x100px (square)
   - Used in: Student cards, Parent dashboard child info

4. teacher-avatar.png
   - Default teacher profile photo placeholder
   - Recommended size: 100x100px (square)
   - Used in: Teacher cards, Teacher dashboard

5. admin-avatar.png
   - Default admin profile photo placeholder
   - Recommended size: 100x100px (square)
   - Used in: Navbar avatar, Admin profile

6. parent-avatar.png
   - Default parent profile photo placeholder
   - Recommended size: 100x100px (square)
   - Used in: Parent dashboard, Parent profile

7. trophy-icon.png
   - Trophy icon for Schoolers brand
   - Recommended size: 40x40px
   - Used in: Logo area, branding

8. school-building.png
   - School building illustration (optional)
   - Recommended size: 300x200px
   - Used in: Background decoration

9. no-data.png
   - Empty state illustration
   - Recommended size: 200x200px
   - Used in: Empty tables, no results found

10. payment-success.png
    - Payment success illustration
    - Recommended size: 200x200px
    - Used in: Fee payment confirmation

NOTE: All images should be in PNG or JPG format.
      For best performance, compress images before use.
      SVG format is also acceptable for icons and logos.

DESIGN THEME:
- Primary Color: #76C442 (lime green)
- Background: white (#ffffff)
- Text: #333333
- Font: Inter, Poppins
